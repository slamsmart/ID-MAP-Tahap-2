Instruksi Export Tile Layer MHI

Untuk: Dimas Syarif Alim, S.Kel — GIS Specialist ID-MAP
Dari: Slamet Abdullah — Ketua Tim ID-MAP

Tujuan: Mendapatkan URL tile layer MHI agar bisa di-overlay langsung di peta Leaflet ID-MAP. Dengan ini, user bisa melihat warna MHI (hijau/kuning/merah) sambil menggambar polygon untuk menghitung carbon stock — semua dalam 1 peta yang sinkron.

Langkah-langkah

Step 1: Buka Google Earth Engine Code Editor
https://code.earthengine.google.com/
Step 2: Buat script baru, lalu paste kode di bawah ini
Step 3: GANTI path asset MHI di baris var mhi = ... sesuai asset yang kamu punya di GEE
Step 4: Klik Run
Step 5: Lihat di panel Console — copy URL tile yang muncul
Step 6: Kirim URL tersebut ke saya (Slamet)

Script GEE (Copy-Paste)

// ============================================================ // Export MHI Jawa Timur sebagai Tile Layer untuk ID-MAP // ============================================================ // Load image MHI // GANTI PATH INI sesuai asset MHI yang ada di akun GEE kamu var mhi = ee.Image("projects/ee-dimassyarifworkspace/assets/MHI_JawaTimur_2024"); // Visualisasi parameter (sama seperti di app MHI) var visParams = { min: 0, max: 100, palette: ['d7191c', 'f5c542', '1a9641'] // Merah - Kuning - Hijau }; // Generate tile URL var mapId = mhi.getMapId(visParams); // Print tile URL ke Console print('=== TILE URL UNTUK ID-MAP ==='); print('Copy URL di bawah ini dan kirim ke Slamet:'); print(mapId.urlFormat); // Verifikasi di peta var jatim = ee.Geometry.Rectangle([110.5, -8.8, 114.8, -6.5]); Map.centerObject(jatim, 9); Map.addLayer(mhi, visParams, 'MHI Jawa Timur');

Output yang Diharapkan

Setelah klik Run, di panel Console akan muncul URL seperti ini:

https://earthengine.googleapis.com/v1/projects/earthengine-legacy/maps/XXXXXXXXXXXXXXXX/tiles/{z}/{x}/{y}

Copy URL tersebut dan kirim ke saya.

Catatan Penting

⚠ URL ini expired setelah ~2 jam.
Untuk testing dan demo hackathon ini cukup. Nanti kalau butuh permanen, kita bisa export sebagai static tiles ke Google Cloud Storage.

Kalau path asset berbeda:

Cek asset kamu di GEE dengan cara:

  1. Klik tab Assets di panel kiri GEE
  2. Cari image MHI yang sudah kamu upload/generate
  3. Klik image tersebut → copy path-nya
  4. Ganti di script: var mhi = ee.Image("PATH_KAMU_DISINI");

Kalau MHI-nya berupa ImageCollection (bukan single Image):

Ganti baris pertama menjadi:

var mhi = ee.ImageCollection("projects/ee-dimassyarifworkspace/assets/MHI_collection") .sort('system:time_start', false) .first(); // Ambil yang terbaru

Kenapa Ini Dibutuhkan?

Saat ini ID-MAP menampilkan MHI via iframe (embed app GEE). Masalahnya:

  • User tidak bisa gambar polygon di atas peta MHI (iframe cross-origin)
  • Saat switch ke mode polygon, posisi peta berubah (tidak sinkron)
  • Kalkulasi luas area jadi tidak akurat

Dengan tile URL, saya bisa overlay MHI langsung di Leaflet → user lihat warna MHI + gambar polygon + hitung carbon stock, semua dalam 1 peta yang sama. Akurat dan seamless.
