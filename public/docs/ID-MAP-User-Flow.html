ID-MAP
User Flow Komprehensif

Platform Integrasi Data Ekosistem Pesisir Berkelanjutan — pemetaan alur pengguna lintas 5 peran (Sahabat, Mitra, Verifikator, Admin, Corporate)
Versi: 1.0 · Tanggal: 26 Mei 2026
Stack: Next.js 14 · Convex · Mayar.id · Cloudinary

1. Ringkasan Eksekutif

ID-MAP adalah platform digital yang mengintegrasikan data ekosistem pesisir Indonesia (mangrove, abrasi, habitat penyu, jaringan Pokmaswas) sekaligus menjadi jembatan antara donatur publik, kelompok masyarakat pelaksana, regulator/DKP, dan korporasi ESG.

Dokumen ini memetakan seluruh alur pengguna pada platform: dari kunjungan publik di landing page, registrasi peran, donasi QRIS, pengelolaan proyek MRV, audit oleh verifikator DKP, hingga pembelian carbon credit oleh korporasi.

5 Peran Utama

  • Sahabat Pesisir — donatur mikro
  • Mitra — Pokmaswas / NGO pelaksana
  • Verifikator — DKP / KKP audit
  • Admin — manajemen sistem
  • Corporate — pembeli carbon credit

Alur Inti

  • Donasi QRIS → sertifikat otomatis
  • Project lifecycle: Draft → Verifikasi → Live
  • MRV cycle: Monitoring → Reporting → Verification
  • Cross-role: donasi memicu funding → memicu MRV

5.2 Donasi QRIS Flow

/donasi-cepat/[projectId]
   │
   ▼
[Pilih nominal]: 10rb · 25rb · 50rb · 100rb · 250rb · 500rb · custom
   │
   ▼
[Klik "Buat QR Pembayaran"]
   │
   ▼
POST /api/payment/create-qris { amount, projectId, userId }
 ├── Sandbox  : dummy paymentId, fallback QR.react
 └── Live     : Mayar.id /qrcode/create → real QR
   │
   ▼
[State: waiting] → poll /api/payment/status setiap 3 detik
   │
   ▼
[Mayar webhook] POST /api/payment/webhook
 → verifyWebhook(rawBody, headers)  [HMAC/Bearer]
 → confirmByPaymentId(paymentId)
   ├── flip paymentStatus: "paid"
   ├── increment fundingRaised di project
   └── auto-insert certificates row (type: "contribution")
   │
   ▼
[State: paid] → CTA "Lihat sertifikat saya"
   │
   ▼
/user/sertifikat → preview + WA/X/copy share

11. Edge Cases & Recovery

SkenarioHandling Saat Ini
Donor scan QR proyek yang sudah Draft/donasi-cepat/[id] cek project === null → screen "Proyek tidak ditemukan" + link /proyek
Webhook gagal / lambatFrontend polling /api/payment/status setiap 3 detik selama state waiting
OTP tidak masuk inboxUI ada warning "cek folder Spam" + tombol Resend (60s cooldown)
Mitra hapus proyek yg sudah punya donasiBelum ada guard — risiko orphan contributions (TODO)
Verifikator edit project tanpa konfirmasi mitraUI checkbox wajib + warning amber
Token sesi expiredgetSession() → redirect /masuk?next=...
Payment di prod tapi MAYAR_API_KEY kosongAuto fallback dummy mode + banner "Mode Demo"
