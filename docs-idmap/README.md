# ID-MAP Documentation Pack

Folder ini berisi ringkasan dokumentasi produk dan teknis ID-MAP v2.0 dalam format Markdown yang siap dipakai untuk Notion, proposal, review juri, atau handover developer.

Scope dokumen ini mengikuti ruang lingkup terbaru:
- Integrasi data pesisir
- Restorasi dan proyek mangrove
- Donasi publik
- KYC dan verifikasi
- MRV digital
- Peta interaktif
- AI chatbot dan analisis mangrove
- Admin dan operasional platform

Di luar scope dokumen ini:
- Corporate Carbon Purchase Flow
- Carbon credit marketplace
- ESG portfolio corporate
- Jual beli tCO2e sebagai marketplace
- Blockchain/tokenisasi karbon

## Daftar Dokumen

| File | Isi |
|---|---|
| `idmap_production_architecture.md` | Arsitektur production, runtime flow, deployment, security, dan observability |
| `idmap_visi_masalah_solusi.md` | Visi, masalah utama, solusi, target user, dan value proposition |
| `idmap_analisis_konsep.md` | Analisis konsep produk, prinsip, modul, flow, dan batasan scope |
| `idmap_tech_stack.md` | Stack teknologi, dependency, alasan pemilihan, env, dan scripts |
| `idmap_roadmap.md` | Roadmap BMAD, epic, sprint, prioritas, dan pilot readiness |
| `idmap_frontend_summary.md` | Ringkasan frontend, halaman, komponen, UX, responsive, dan accessibility |
| `payment_gateway_integration.md` | Integrasi payment Mayar.id untuk donasi QRIS/invoice dan webhook |
