# ID-MAP Visi, Masalah, dan Solusi

## 1. Visi

ID-MAP memiliki visi menjadi platform digital terpadu untuk integrasi data pesisir Indonesia, pengelolaan proyek restorasi mangrove, pendanaan berbasis donasi publik, verifikasi MRV, dan analisis berbasis AI.

Fokus awal platform adalah memperkuat transparansi dan operasional proyek ekosistem pesisir, bukan marketplace karbon corporate.

## 2. Misi

1. Mengintegrasikan data pesisir yang tersebar menjadi satu platform yang mudah diakses.
2. Membantu mitra dan komunitas mengelola proyek mangrove secara lebih rapi.
3. Memudahkan masyarakat berdonasi dan melihat dampak kontribusinya.
4. Menyediakan alur KYC, MRV, dan verifikasi proyek yang lebih tertata.
5. Menyajikan peta interaktif untuk mangrove, abrasi, penyu, dan Pokmaswas.
6. Menggunakan AI untuk membantu edukasi, analisis, dan dukungan operasional.

## 3. Masalah Utama

| ID | Masalah | Dampak |
|---|---|---|
| P1 | Data mangrove, abrasi, penyu, dan Pokmaswas tersebar | Sulit membuat keputusan cepat dan akurat |
| P2 | Proyek restorasi sulit menunjukkan progress secara transparan | Donatur dan pemangku kepentingan sulit percaya |
| P3 | Donasi lingkungan sering tidak terlihat dampaknya | Partisipasi publik rendah |
| P4 | MRV masih banyak dilakukan manual | Verifikasi lambat dan rawan error |
| P5 | KYC mitra belum terpusat | Risiko validitas organisasi/proyek |
| P6 | Konten edukasi dan data publik sulit diperbarui | Platform cepat usang |
| P7 | Komunitas lokal dan Pokmaswas belum punya kanal digital yang kuat | Aktivitas lapangan kurang terlihat |

## 4. Akar Masalah

1. Data berada di banyak file, dashboard, atau dokumen terpisah.
2. Proyek lapangan sering mengandalkan laporan manual.
3. Sistem donasi tidak selalu terhubung ke data proyek.
4. Verifikasi membutuhkan koordinasi antar pihak.
5. Tim operasional butuh dashboard, bukan hanya website publik.
6. Akses informasi ekosistem pesisir masih terlalu teknis bagi publik.

## 5. Solusi Produk

ID-MAP menyelesaikan masalah tersebut melalui modul berikut:

| Modul | Solusi |
|---|---|
| Landing & Edukasi | Menjelaskan ekosistem pesisir dan program ID-MAP |
| Project Registry | Menyimpan data proyek, lokasi, status, progress, pendanaan |
| Donation Engine | Donasi QRIS/invoice yang terhubung ke proyek |
| Sahabat Dashboard | Riwayat kontribusi, dampak, dan sertifikat |
| Mitra Dashboard | Submit KYC, proyek, pendanaan, dan MRV |
| Verifikator Dashboard | Review data, MRV, GIS, dan konten publik |
| Admin Dashboard | Operasional user, proyek, KYC, transaksi, laporan |
| Interactive Map | Visualisasi mangrove, abrasi, penyu, Pokmaswas |
| AI Chat & Analysis | Edukasi, bantuan operasional, dan analisis mangrove |

## 6. Target User

### Sahabat

Masyarakat umum yang ingin berkontribusi terhadap restorasi mangrove melalui donasi kecil atau berkala.

Kebutuhan:
- Donasi mudah.
- Dampak terlihat.
- Sertifikat kontribusi.
- Informasi proyek mudah dipahami.

### Mitra

NGO, komunitas, Pokmaswas, atau pengelola proyek restorasi.

Kebutuhan:
- KYC organisasi.
- Manajemen proyek.
- Pendanaan.
- Laporan MRV.
- Visibilitas publik.

### Mitra Facilitator

Organisasi pendamping seperti NGO, akademisi, koperasi, atau pemerintah daerah.

Kebutuhan:
- Mendampingi proyek komunitas.
- Menjaga kualitas data.
- Membantu koordinasi dan validasi.

### Verifikator

Pihak yang melakukan verifikasi data, laporan, dan informasi lapangan.

Kebutuhan:
- Queue review.
- Data proyek dan MRV.
- Input/update data peta.
- Konten publik yang dapat diperbarui.

### Admin

Operator platform ID-MAP.

Kebutuhan:
- Kontrol user.
- Review KYC.
- Monitoring proyek.
- Monitoring transaksi donasi.
- Konfigurasi platform.
- Laporan operasional.

## 7. Value Proposition

| Audience | Value Proposition |
|---|---|
| Sahabat | Donasi mangrove yang mudah, transparan, dan berdampak |
| Mitra | Satu dashboard untuk proyek, KYC, pendanaan, dan MRV |
| Verifikator | Alur review data dan laporan yang lebih terstruktur |
| Admin | Kontrol operasional platform dalam satu sistem |
| Publik | Akses edukasi dan data pesisir yang lebih mudah dipahami |

## 8. Product Positioning

ID-MAP v2.0 diposisikan sebagai:

> Platform digital multi-role untuk integrasi data pesisir, pendanaan restorasi mangrove, donasi publik, verifikasi MRV, peta interaktif, dan analisis AI, dengan fokus awal pada ekosistem mangrove Indonesia.

## 9. Scope Saat Ini

In scope:
- Public information and education.
- Public donation.
- Project management.
- KYC.
- MRV.
- Interactive map.
- AI support.
- Multi-role dashboards.
- Admin operations.

Out of scope:
- Corporate Carbon Purchase Flow.
- Carbon credit marketplace.
- ESG portfolio corporate.
- Jual beli tCO2e.
- Blockchain/tokenisasi karbon.
- Native mobile app.
- IoT sensor real-time.
